# 🐵 Monkey Escape

A 3D browser obby: play a monkey escaping a chimpanzee across six treetop stages.
Built with [Three.js](https://threejs.org). No build step — it's static files.

**Play:** open `index.html` via any static server, or the GitHub Pages link.

## Controls
- **WASD / Arrows** — move · **Space** — jump (×2 = Monkey Flip) · hold **Space** — glide
- **Shift** — vine dash · **E** — throw 🍌 to stun the chimp · **Esc** — pause · **M** — mute
- Swing on hanging vines, bounce off mushrooms, grab bananas, reach the banana tree to clear a stage.
- Touch devices get on-screen controls.

## Run locally
```
python3 -m http.server 8123
# open http://localhost:8123
```
